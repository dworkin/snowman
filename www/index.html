Project Snowman
The Project Snowman application is a capture the flag game built on top of the Project Darkstar stack.
Source code
The publicly accessible Subversion repository contains all of the source code for the Project Snowman.
Issue tracker
Issuezilla is used to track all issues with Project Snowman including defects, enhancements, and outstanding tasks.
Mailing Lists
Subversion commit notifications and Issue tracker issue change notifications are sent to separate mailing lists for Project Snowman.
